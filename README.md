# NumRush

NumRush is a lightweight browser game where numbers move from right to left in the URL bar. Match incoming numbers by selecting the correct digit before they reach the catcher on the left.

## How to Play

* Press any digit (0–9) to start.
* Incoming numbers appear in the URL bar.
* Set your catcher digit to match approaching numbers.
* Use:

  * Number keys (0–9) to directly select a digit.
  * Arrow Up / Arrow Down to cycle through digits.
* Matching a number increases your score.
* Missing a number costs a life.
* The game ends when all 3 lives are lost.

## Difficulty

* Every 5 successful catches increase the level.
* Each level makes numbers spawn 10% faster.
* Spawn speed is capped to keep the game playable.

## Technologies

* HTML
* CSS
* JavaScript (Vanilla)

## Running the Game

Open `numrush.html` in any modern web browser.
